# PROJECT_CONTEXT

# SMARTKITCHEN

## Project Overview

SMARTKITCHEN is an AI-powered cooking assistant designed to help users decide what to cook based on available ingredients.

## Target Users

- International students
- People living alone
- Beginner cooks
- Users who don't know what to cook

## MVP Goal

Help users generate suitable recipes using ingredients they already have.

## Tech Stack

Frontend:

- React
- Vite

Backend:

- Node.js
- Express.js

Database:

- SQLite

AI:

- DeepSeek API

## MVP Features

- User Registration
- User Login
- Ingredient Management
- Cuisine Selection
- AI Recipe Recommendation
- Recipe Detail Page
- Save Recipe

## Future Features

- Food Recognition
- Expiry Date Recognition
- Shopping List
- Calorie Analysis
- Community Features
- AI Chatbot
