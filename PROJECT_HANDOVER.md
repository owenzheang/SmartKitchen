# Project Overview

ChefSpark is an AI-powered mobile-first recipe assistant.

The project started as a university portfolio project but has evolved into a real product that is intended for public use rather than only serving as a demonstration.

Current philosophy:

> Build an AI product that solves a real problem with excellent UX, instead of simply wrapping ChatGPT.

# Important Notes For Future AI Assistants

Do NOT redesign the entire architecture.

Preserve existing project structure.

Avoid unnecessary frameworks.

Prefer incremental improvements.

Maintain mobile-first design.

Always explain proposed changes before implementation.

Keep UI modern and premium.

Whenever proposing new features, ask:

"Why wouldn't the user just use ChatGPT?"

Only implement features that provide clear workflow advantages over generic AI chat.
